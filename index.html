Programación

Programación

La programación consiste en diseñar instrucciones para que una computadora realice tareas específicas. Un programa es una secuencia ordenada de instrucciones escritas en un lenguaje de programación y ejecutadas por la computadora.

Introducción a las Computadoras

Conceptos básicos sobre hardware, software, la arquitectura de Von Neumann, la CPU, los tipos de memoria (RAM, ROM y Caché), dispositivos de entrada y salida, almacenamiento, bits y bytes, sistemas de numeración, representación de texto, imágenes y sonido, además de una introducción a redes, Internet, el modelo cliente-servidor y la World Wide Web. Basado principalmente en Joyanes Aguilar (2020).

Algoritmos

Un algoritmo es una secuencia finita y ordenada de pasos que permiten resolver un problema. Se abordan su definición, características, pseudocódigo, diagramas de flujo, complejidad, y las etapas de entrada, proceso y salida, acompañados de ejemplos como calcular un promedio, buscar un número, ordenar una lista o encontrar el mayor elemento de un conjunto.

Programación en Python

Fundamentos del lenguaje Python: variables, tipos de datos, operadores, condicionales, bucles, funciones, listas, tuplas, diccionarios, conjuntos, manejo de archivos, excepciones y una introducción a la Programación Orientada a Objetos, con un ejemplo sencillo para cada tema.

Estructuras de Datos

Explicación de las principales estructuras de datos utilizadas en programación: arrays, listas enlazadas, pilas, colas, árboles, grafos y tablas hash, cada una implementada con ejemplos en Python.

Desarrollo Web

Desarrollo Web

Esta categoría reúne los fundamentos para construir páginas y aplicaciones web, desde la estructura y el estilo hasta la interactividad y el control de versiones.

HTML, CSS y JavaScript

Estructura de páginas web con HTML, estilos y diseño visual con CSS, e interactividad del lado del cliente con JavaScript, incluyendo ejemplos de formularios, tablas y navegación.

DOM y Responsive Design

Manipulación del DOM (Document Object Model) para modificar una página de forma dinámica, y principios de diseño responsivo para que un sitio se vea correctamente en distintos tamaños de pantalla.

Próximamente.

Git, GitHub y GitHub Pages

Control de versiones con Git, colaboración y almacenamiento de proyectos en GitHub, y publicación de sitios web estáticos utilizando GitHub Pages.

Ciberseguridad

Ciberseguridad

La ciberseguridad es el conjunto de técnicas, procesos y herramientas destinadas a proteger sistemas, redes, aplicaciones y datos frente a amenazas informáticas. Se abordan conceptos básicos como confidencialidad, integridad, disponibilidad, autenticación, autorización, amenazas, vulnerabilidades y riesgos.

Malware

Tipos de software malicioso: virus, worms, troyanos, spyware, keyloggers, rootkits, botnets, ransomware y malware fileless, junto con ejemplos en Python sobre simulación de análisis de archivos, búsqueda de cadenas sospechosas y cálculo de hashes.

Próximamente.

Ingeniería Inversa de Malware

Análisis estático, dinámico e híbrido de muestras maliciosas, técnicas de persistencia y evasión (packing, polimorfismo, metamorfismo), malware fileless, LOLBAS, malware desarrollado en Rust, ransomware, automatización del desempaquetado e Inteligencia Artificial aplicada al análisis, usando herramientas como Ghidra, IDA Pro, x64dbg, PEStudio, Wireshark y Volatility.

Hacking Ético

Fundamentos del pentesting y sus fases: reconocimiento, escaneo, enumeración, explotación, post-explotación y elaboración de informes, con ejemplos en Python sobre resolución de nombres de dominio, escaneo básico de puertos, consultas DNS y uso de sockets.

Próximamente.

Análisis Forense

Conceptos de evidencia digital y cadena de custodia, adquisición de evidencia, análisis de memoria y discos, y recuperación de archivos, con ejemplos en Python para calcular y comparar hashes y leer metadatos básicos de un archivo.

Próximamente.

Inteligencia Artificial

Inteligencia Artificial

Introducción a la Inteligencia Artificial, el Machine Learning, el Deep Learning, las redes neuronales, la IA generativa, la visión por computadora y el procesamiento del lenguaje natural, junto con sus aplicaciones en medicina, finanzas, educación, ciberseguridad y robótica.

Incluye ejemplos en Python de clasificación sencilla, predicción con un modelo básico, procesamiento de texto y uso introductorio de bibliotecas como scikit-learn.

Analizador Básico de Archivos Sospechosos

Proyecto que combina programación en Python y conceptos de ciberseguridad: un script que calcula el hash SHA-256 de un archivo, extrae texto legible de su contenido, busca palabras clave, URLs e IPs sospechosas, asigna un nivel de riesgo (bajo, medio o alto) y genera un reporte visual en HTML.

Ingeniería Inversa de Malware: Técnicas Modernas de Análisis, Persistencia y Detección de Amenazas Avanzadas

La ingeniería inversa de malware es una técnica fundamental dentro de la ciberseguridad moderna. Permite comprender el funcionamiento interno de amenazas avanzadas, identificar mecanismos de persistencia, analizar ransomware, estudiar malware fileless y mejorar las capacidades de defensa frente a ataques sofisticados.

En este artículo se exploran conceptos como:

  • Análisis estático.
  • Análisis dinámico.
  • Análisis híbrido.
  • Persistencia.
  • Técnicas de evasión.
  • LOLBAS.
  • Malware desarrollado en Rust.
  • Ransomware.
  • Inteligencia Artificial aplicada al análisis.
  • Tendencias actuales de ciberseguridad.
