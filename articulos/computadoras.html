¿Qué es una Computadora?

Una computadora es una máquina capaz de recibir datos, procesarlos siguiendo un conjunto de instrucciones y producir un resultado. Este proceso se resume en tres pasos básicos: entrada, procesamiento y salida.

Según Joyanes Aguilar (2020), una computadora está formada por dos grandes componentes que trabajan de forma conjunta: el hardware y el software.

Hardware y Software

Hardware

Es la parte física de la computadora, es decir, todos los componentes que se pueden tocar: el procesador, la memoria, el teclado, el monitor, entre otros.

Software

Es el conjunto de programas e instrucciones que le indican al hardware qué hacer. Se divide principalmente en dos tipos:

  • Software de sistema (por ejemplo, el sistema operativo).
  • Software de aplicación (programas que usa el usuario final).

Arquitectura de Von Neumann

La mayoría de las computadoras actuales siguen un modelo propuesto por John Von Neumann, en el cual tanto los datos como las instrucciones se almacenan en la misma memoria.

Componentes principales

  • Unidad Central de Procesamiento (CPU).
  • Memoria principal.
  • Dispositivos de entrada y salida.
  • Buses de comunicación.

Este modelo permite que un mismo programa pueda ser modificado en memoria, lo que hace que las computadoras sean máquinas flexibles y programables.

La CPU

La CPU (Unidad Central de Procesamiento) es el componente encargado de ejecutar las instrucciones de los programas. Se le suele llamar el "cerebro" de la computadora.

Unidad de Control

Se encarga de dirigir el funcionamiento de la CPU, indicando qué instrucción debe ejecutarse y en qué momento.

Unidad Aritmética y Lógica (ALU)

Es la parte de la CPU responsable de realizar operaciones matemáticas (sumas, restas, etc.) y operaciones lógicas (comparaciones, condiciones).

Memoria RAM, ROM y Caché

Memoria RAM

Es la memoria principal donde se almacenan temporalmente los datos y programas que la CPU está utilizando en un momento dado. Se borra al apagar el equipo.

Memoria ROM

Es una memoria de solo lectura que contiene instrucciones básicas necesarias para el arranque de la computadora. No se borra al apagar el equipo.

Memoria Caché

Es una memoria muy rápida y de menor tamaño, ubicada entre la CPU y la RAM, cuyo objetivo es acelerar el acceso a los datos usados con mayor frecuencia.

Dispositivos de Entrada y Salida

Dispositivos de Entrada

Permiten ingresar información a la computadora. Ejemplos: teclado, mouse, escáner, micrófono.

Dispositivos de Salida

Permiten mostrar o entregar el resultado del procesamiento. Ejemplos: monitor, impresora, altavoces.

Almacenamiento

A diferencia de la RAM, los dispositivos de almacenamiento guardan la información de forma permanente, incluso cuando la computadora se apaga.

  • Discos duros (HDD).
  • Unidades de estado sólido (SSD).
  • Memorias USB.
  • Almacenamiento en la nube.

Bits, Bytes y Sistemas de Numeración

La computadora representa toda la información internamente utilizando únicamente dos valores: 0 y 1. A cada uno de estos valores se le llama bit.

Un conjunto de 8 bits forma un byte, la unidad básica utilizada para medir la cantidad de información.

Sistemas de numeración

  • Binario (base 2): utilizado internamente por la computadora.
  • Decimal (base 10): el sistema que usamos habitualmente.
  • Hexadecimal (base 16): utilizado frecuentemente en programación y para representar direcciones de memoria.

Representación de Texto, Imágenes y Sonido

Texto

Cada carácter (letras, números, símbolos) se representa mediante un código numérico, como el estándar ASCII o Unicode.

Imágenes

Una imagen digital está formada por una matriz de puntos llamados píxeles, donde cada uno almacena información sobre su color.

Sonido

El sonido, al ser una señal analógica, se convierte a valores digitales mediante un proceso llamado muestreo, para que pueda ser almacenado y reproducido por la computadora.

Redes, Internet y la World Wide Web

Redes

Una red es un conjunto de computadoras conectadas entre sí con el objetivo de compartir información y recursos.

Internet

Internet es una red global que conecta millones de redes y dispositivos alrededor del mundo, permitiendo el intercambio de información.

Modelo Cliente-Servidor

Es un modelo en el que un cliente (por ejemplo, un navegador) solicita información, y un servidor responde a esa solicitud entregando los datos requeridos.

World Wide Web (WWW)

Es un sistema de documentos interconectados mediante hipervínculos, accesibles a través de Internet utilizando un navegador web.

Ejemplos en Python

A continuación se muestran algunos ejemplos sencillos relacionados con los conceptos vistos en este artículo.

Mostrar información básica del sistema 

import platform

print("Sistema operativo:", platform.system())
print("Versión:", platform.version())
print("Arquitectura:", platform.machine())

Este ejemplo permite obtener información básica sobre el sistema operativo y la arquitectura del equipo donde se ejecuta el programa.

Leer un archivo de texto 

with open("ejemplo.txt", "r") as archivo:
    contenido = archivo.read()

print(contenido)

Muestra cómo un programa puede acceder al almacenamiento para leer datos guardados de forma permanente.

Trabajar con memoria mediante variables 

dato = 10
print("Valor almacenado en memoria:", dato)

dato = dato + 5
print("Nuevo valor almacenado en memoria:", dato)

Las variables representan espacios de memoria donde un programa guarda y modifica información mientras se ejecuta.

Conversión entre unidades (bytes a kilobytes) 

bytes_totales = 5000

kilobytes = bytes_totales / 1024

print("Bytes:", bytes_totales)
print("Kilobytes:", kilobytes)

Permite comprender cómo se relacionan las distintas unidades utilizadas para medir el almacenamiento y la memoria.

Dato Interesante

La primera computadora electrónica de propósito general, conocida como ENIAC, ocupaba una habitación completa y pesaba cerca de 27 toneladas. Hoy en día, un simple teléfono celular tiene mucha más capacidad de procesamiento que aquella máquina.
