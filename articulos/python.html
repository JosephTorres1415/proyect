¿Qué es Python?

Python es un lenguaje de programación de alto nivel, interpretado y de sintaxis sencilla, muy utilizado en automatización, análisis de datos, desarrollo web e inteligencia artificial. Su legibilidad lo convierte en una excelente opción para aprender a programar.

Variables y Tipos de Datos

Una variable es un espacio en memoria donde se almacena un valor que puede cambiar durante la ejecución del programa. En Python no es necesario declarar el tipo de dato de forma explícita.

Tipos de datos básicos

  • int: números enteros.
  • float: números decimales.
  • str: cadenas de texto.
  • bool: valores verdadero o falso.
nombre = "Joseph"
edad = 20
promedio = 15.5
es_estudiante = True

print(nombre, edad, promedio, es_estudiante)

Operadores

Operadores aritméticos 

a = 10
b = 3

print(a + b)
print(a - b)
print(a * b)
print(a / b)
print(a % b)

Operadores de comparación 

print(a > b)
print(a == b)
print(a != b)

Condicionales

Permiten que el programa tome decisiones y ejecute distintas acciones según se cumpla o no una condición. 

nota = 14

if nota >= 11:
    print("Aprobado")
else:
    print("Desaprobado")

Bucles

Permiten repetir un conjunto de instrucciones varias veces.

Bucle for 

for numero in range(1, 6):
    print("Número:", numero)

Bucle while 

contador = 0

while contador < 3:
    print("Contador:", contador)
    contador += 1

Funciones

Una función es un bloque de código reutilizable que realiza una tarea específica y puede recibir datos de entrada y devolver un resultado. 

def sumar(a, b):
    return a + b

resultado = sumar(4, 7)
print("Resultado:", resultado)

Listas y Tuplas

Listas

Colecciones ordenadas y modificables de elementos. 

frutas = ["manzana", "pera", "uva"]
frutas.append("naranja")

print(frutas)

Tuplas

Colecciones ordenadas, pero a diferencia de las listas, no se pueden modificar una vez creadas. 

coordenadas = (10, 20)

print(coordenadas)

Diccionarios y Conjuntos

Diccionarios

Almacenan información en pares de clave y valor. 

estudiante = {
    "nombre": "Joseph",
    "edad": 20
}

print(estudiante["nombre"])

Conjuntos

Colecciones que no permiten elementos repetidos. 

numeros = {1, 2, 2, 3, 3, 3}

print(numeros)

Archivos

Python permite leer y escribir archivos de texto para almacenar información de forma permanente. 

with open("notas.txt", "w") as archivo:
    archivo.write("Apuntes de Python")

with open("notas.txt", "r") as archivo:
    print(archivo.read())

Excepciones

Permiten manejar errores que ocurren durante la ejecución del programa, evitando que este se detenga de forma inesperada. 

try:
    resultado = 10 / 0
except ZeroDivisionError:
    print("No se puede dividir entre cero")

Programación Orientada a Objetos

La Programación Orientada a Objetos (POO) organiza el código en clases y objetos, que representan entidades con atributos (datos) y métodos (comportamientos). 

class Estudiante:
    def __init__(self, nombre, edad):
        self.nombre = nombre
        self.edad = edad

    def saludar(self):
        print(f"Hola, soy {self.nombre}")

alumno = Estudiante("Joseph", 20)
alumno.saludar()

Este paradigma facilita organizar programas más grandes, agrupando datos y comportamientos relacionados dentro de una misma estructura.

Dato Interesante

El nombre "Python" no proviene de la serpiente, sino del grupo de comedia británico Monty Python. Su creador, Guido van Rossum, era fan del programa y buscaba un nombre corto y memorable para el lenguaje.
